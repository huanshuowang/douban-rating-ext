# 这豆瓣评分多少？

**看剧 / 看电影 / 刷解说 / 逛搜索页，点一下就知道豆瓣多少分。** 读取当前网页的片名，直接显示**豆瓣评分**并一键跳转到豆瓣条目页。

**任何带片名的页面都能用**——Netflix、YouTube、爱奇艺、腾讯视频、Bilibili、Google / Bing 搜索页……哪儿有片名就查哪儿。国内、海外都行；页面上安静无干扰；不直接爬豆瓣搜索接口，**没有封 IP 的风险**。

---

## ✨ 功能

- **点图标即查**：读取当前网页的片名，自动去掉「站名 / Watch / 官方网站」等外壳清洗成片名（各大流媒体、搜索结果页、影视解说标题都认得）。
- **直接显示评分**：大号数字 + 「在豆瓣查看」直达条目页。
- **国内 / 海外自适应**：同时问 Google / Bing / 百度，谁先返回评分就用谁（海外走 Google，国内 Google 超时自动落到百度）。
- **手动兜底**：识别不准时可改关键词回车重查，或用「百度 / 豆瓣直搜 / Google / Bing」按钮手动搜。
- **本地缓存 7 天**：同一部片只查一次。

## 🔒 获取评分的方式安全吗

- **豆瓣的搜索接口对海外 IP 限流极狠**（常见「错误103 / 搜索访问太频繁」），所以本插件**不碰豆瓣搜索**。
- 评分优先从**搜索引擎摘要**里读（评分本来就印在结果里）。
- 只有当摘要里没有评分、但已经通过搜索拿到了**豆瓣条目 id** 时，才会去读**那一个条目页**取分——就是你点绿色按钮跳转的那种页面，一部片只读一次并缓存。豆瓣接触量 ≈ 你手动点开一次，远离被限流的搜索接口。
- 所有网络请求**只在你打开插件那一刻**发生，页面浏览期间零请求。


## 🔒 隐私政策

本插件不运营自己的服务器，不出售或共享用户数据。查询只会在你点击插件图标时发生，必要的片名关键词会直接从你的浏览器发送给搜索引擎或豆瓣页面以获取评分。完整隐私政策见 [`privacy.html`](privacy.html)。

## 📦 安装（开发者模式加载）

1. 打开 `chrome://extensions`
2. 右上角打开 **开发者模式**
3. 点 **加载已解压的扩展程序**，选择本文件夹 `douban-rating-ext`
4. 打开任意带片名的页面（Netflix / YouTube / 爱奇艺 / Bilibili / Google 搜索…），点右上角插件图标

> 更新代码后：回到 `chrome://extensions` 点该插件的刷新 ↻；如果改动了 `manifest.json`（比如换图标/权限），建议**移除后重新加载**。

## 🎨 更换图标

默认图标是 `icons/score/`（绿底白字「8.9」），共 4 个尺寸：16 / 32 / 48 / 128 px。
想换成别的，直接替换 `icons/score/` 下这 4 个同名 PNG，或新建一套图标目录后改 `manifest.json` 里的图标路径即可。

## 🗂 文件说明

| 文件 | 作用 |
| --- | --- |
| `manifest.json` | 插件清单（权限、图标、popup 入口） |
| `popup.html` / `popup.js` | 点图标弹出的面板 + 全部查询逻辑 |
| `icons/score/` | 图标「8.9」（16/32/48/128 px） |
| `index.html` | 独立介绍页（双击即可在浏览器打开预览） |

## ⚠️ 已知限制

- 解析搜索摘要是**尽力而为**：搜索引擎偶尔改版或某条摘要没放评分，会显示「未找到」，用手动按钮兜底即可。
- 片名识别对多数「流媒体详情页 / 搜索结果 / 影视正片 / UGC 影视解说」有效；番剧分季、纪录片、竖屏短视频、冷门译名等特殊标题可能需要手动改关键词。
- 仅供个人自用，请勿用于大规模抓取或商业用途。
