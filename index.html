它能做什么

点图标即查读取当前网页的片名，自动去掉站名/前后缀清洗成片名
大号评分 + 直达显示豆瓣分数，一键跳转豆瓣条目页
三家竞速Google / Bing / 百度谁先出评分用谁
随时改词重查弹窗自带搜索框，识别不准回车即可重查 · 缓存 7 天

所有网页 · 随时都能用

插件读的是当前页面的片名——哪儿有片名就能查。已验证可用：Bilibili、Netflix、YouTube、Google / Bing 搜索结果页 等。片名识别不准时，弹窗顶部的搜索框可随时改关键词回车重查。

获取评分的方式安全吗

豆瓣的搜索接口对海外 IP 限流极狠，所以本插件不碰豆瓣搜索。评分优先从搜索引擎摘要里读；只有当摘要没评分、却已经拿到豆瓣条目 id 时，才去读那一个条目页取分。

所有网络请求只在你打开插件那一刻发生，浏览页面时零请求。豆瓣接触量 ≈ 你手动点开一次。

安装

  1. 点上方 前往 Chrome 应用商店安装，进入这个 Chrome extension 的商店页
  2. 点右上角 「添加至 Chrome」，在弹窗里确认 「添加扩展程序」
  3. （可选）点 Chrome 工具栏的拼图图标，把「这豆瓣评分多少？」固定到工具栏，方便随时点
  4. 打开任意带片名的页面（Netflix / YouTube / 爱奇艺 / Bilibili / Google 搜索…），点右上角插件图标
数据来自公开搜索结果与豆瓣页面
